# Online Bookstore - Sprint 1

## Project Scope
The Online Bookstore project is a web application where users can browse books, search for books, view book details, register/login, and add books to a shopping cart.

## Functional Requirements
- FR1: User can view the home page
- FR2: User can browse books
- FR3: User can search for books
- FR4: User can view book details
- FR5: User can register an account
- FR6: User can log in
- FR7: User can add books to the cart
- FR8: User can view the cart

## Non-Functional Requirements
- NF1: The website should be easy to use
- NF2: The website should be responsive
- NF3: Pages should load quickly
- NF4: User data should be secure
- NF5: The system should be reliable
